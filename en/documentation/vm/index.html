    • Once KJOS has been downloaded, install Virtual Box or any other virtual machine emulation software.

    Create a new virtual machine.

    Configure the virtual machine as below.

    ⚠️ Set the Windows version according to the chosen OS:

    KJOS ▶️ Windows 11

    KJOS-Lite ▶️ Windows 10

    ⚠️ WARNING ⚠️

    For configuration, put at least the minimum configuration of the chosen KJOS version.

    EFI mode must be disabled for KJOS-Lite!

    For the rest, leave the default settings and finish.

    Then click on configuration.

    Load ISO into virtual machine disks in:

    Storage -> Empty -> Optical drive -> Choose a disk file -> OK

    Start the VM and click on the window that has just opened.

    ⚠️ When the machine boots up, you'll need to press the 'Enter' key several times to start the installation, the windows icon (or virtual box) will be displayed.

    The installation wizard will start and display the menu below.

    Click Next and leave all settings at default. Windows will begin to install, this may take a few minutes.

    The VM will reboot numerous times and may remain frozen for a few seconds, don't worry, everything is normal.

    Once installation is complete, a local account creation page will appear.
