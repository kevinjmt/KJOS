What is KJOS ?

Learn more about KJ's Windows 11 custom ISO

KJOS is a free, open-source and ultra-lightweight operating system based on Windows 11, while removing all the problems associated with it.

Everyone complains about the descent into hell that Microsoft is causing with its OSes, including Windows 11.

To cite a few examples: Telemetry, data collection, abundant and ubiquitous ads, sponsored applications, Windows 11 hardware limitations (TPM, Secure Boot...), hidden services etc.

KJOS eliminates all these problems and offers a simple, fluid OS that's accessible on almost any device!

KJOS is based on Windows 11 Pro, which means that if you have a Windows 11 Pro license, your system will be activated automatically!

On the other hand, if you'd like to install KJOS-Lite (based on Windows IoT LTSC, an official, optimized vesion of Windows 11) or if you don't have a Windows 11 Pro key, you can always buy a license online or get your own. Several methods are explained in the documentation.
